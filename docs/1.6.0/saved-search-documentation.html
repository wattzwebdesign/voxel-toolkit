Saved Search

The Saved Search function lets users save their search filters and quickly reload them later. Users can also receive notifications when new posts match their saved searches.

Overview

Visitors often return to your site looking for the same type of content. With Saved Search, they can save their filter combinations and reload them with one click. Even better, they can opt to receive notifications when new content matches their criteria.

Key Features

  • Save Search button in Voxel search form widget
  • Saved Search widget displays saved searches in grid layout
  • Load Search dropdown for quick filter application
  • Auto-apply option for returning visitors
  • Notifications for new matching posts
  • Per-search notification preferences
  • localStorage persistence

Setup Instructions

Step 1: Enable the Function

  1. Navigate to Voxel Toolkit > Functions in your WordPress admin
  2. Find Saved Search in the list
  3. Toggle the switch to enable the function
  4. Click Save Changes

Once enabled, the Save Search button will automatically appear in Voxel's search form widget.

Step 2: Add Saved Search Widget (Optional)

To display saved searches on user profile or account pages:

  1. Edit the page in Elementor
  2. Add the Saved Search (VT) widget
  3. Configure grid layout and styling
  4. Save the page

Save Search Button

The button appears in the Voxel search form when filters are active.

Saving a Search

  1. User applies search filters
  2. User clicks the Save Search button
  3. A popup appears to name the search
  4. User can enable/disable notifications
  5. Search is saved to their account

Load Search Dropdown

A dropdown in the search form lets users quickly apply saved searches:

  • Lists all saved searches for this search form
  • One-click to apply all filters from that search
  • Shows filter summary for each saved search

Saved Search Widget

Displays all saved searches in a grid layout.

Widget Features

  • Grid layout with configurable columns
  • Search name and filter summary
  • Click to go to search with filters applied
  • Delete button to remove saved searches
  • Notification toggle per search

Auto-Apply Feature

Users can set a saved search to auto-apply when they return to the search page:

  • Enable auto-apply on a saved search
  • When they visit the search page, those filters are automatically applied
  • Only one search can be set to auto-apply at a time
  • Uses localStorage for persistence across sessions

Notifications

When new posts match a user's saved search, they can receive notifications:

Notification Types

  • In-App - Notification appears in Voxel's notification system
  • Email - Email sent to the user
  • SMS - Text message (if SMS Notifications enabled)

Configuring Notifications

  1. Go to Voxel > App Events
  2. Find Saved Search: New Match
  3. Configure email template and delivery options

User Controls

Users can enable or disable notifications per saved search:

  • Toggle in the save dialog when creating
  • Toggle in the Saved Search widget
  • Notifications are off by default

Use Cases

Job Seekers

Save search filters for jobs in their field and location, get notified when new jobs are posted.

Property Hunters

Save searches for properties in their price range and preferred neighborhoods.

Marketplace Buyers

Save searches for specific products or categories they're interested in.

Event Attendees

Save searches for events in their area or of specific types.

Storage

Saved searches are stored in:

  • Logged-in users - Database (syncs across devices)
  • Guests - localStorage (browser only)

When a guest logs in, their localStorage searches can be migrated to their account.

Troubleshooting

Save Button Not Appearing

  • Check that the function is enabled
  • Apply at least one filter first (button shows when filters are active)
  • Verify you're using Voxel's search form widget

Saved Searches Not Persisting

  • For guests, check localStorage is enabled
  • Try logging in to save searches to your account
  • Check browser privacy settings

Notifications Not Working

  • Verify notifications are enabled on the saved search
  • Check App Event is configured in Voxel
  • Verify WordPress is sending emails

Requirements

  • Voxel Theme (active)
  • Voxel Toolkit plugin (enabled)
  • Voxel search form widget in use
