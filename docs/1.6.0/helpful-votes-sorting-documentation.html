Helpful Votes Sorting

The Helpful Votes Sorting function adds new sort options to Voxel search forms based on Article Helpful widget votes. Display your most helpful content first by sorting by yes votes, no votes, or total engagement.

Overview

If you use the Article Helpful widget to let users vote on whether content was helpful, this function lets visitors sort your content by those votes. Show the most helpful articles first, or find disputed content that may need improvement.

Key Features

  • Sort by most helpful (yes votes)
  • Sort by most disputed (no votes)
  • Sort by total engagement (all votes)
  • Works with Voxel's search form Order By filter
  • Dynamic tags for vote timestamps

Setup Instructions

Step 1: Enable the Function

  1. Navigate to Voxel Toolkit > Functions in your WordPress admin
  2. Find Helpful Votes Sorting in the list
  3. Toggle the switch to enable the function
  4. Click Save Changes

Step 2: Add Order By Filter

  1. Go to Voxel > Post Types
  2. Edit the post type that uses Article Helpful
  3. Go to Search > Filters
  4. Add or edit the Order By filter
  5. The new sort options will be available automatically
  6. Save the post type

Use Cases

Knowledge Base

Let users sort articles by helpfulness to find the best answers first.

FAQ Pages

Show the most helpful FAQs at the top of your list.

Tutorial Sites

Highlight tutorials that users find most valuable.

Content Moderation

Use "Most Disputed" sorting to find content that may need updating or review.

Recently Helpful Badge

Use the timestamp tags with Elementor visibility rules to show badges like "Recently marked helpful" on articles that received a yes vote within the last week.

Troubleshooting

Sort Options Not Appearing

  • Check that the function is enabled
  • Verify the Order By filter is added to the post type
  • Clear caching and refresh the frontend

Sorting Not Working

  • Ensure posts have Article Helpful vote data
  • Check that the Article Helpful widget has been enabled
  • Verify votes have been cast on posts

Timestamp Tags Empty

  • The post may not have received that type of vote yet
  • Timestamps are only recorded after enabling the function

Requirements

  • Voxel Theme (active)
  • Voxel Toolkit plugin (enabled)
  • Article Helpful widget in use
  • Order By filter on the post type
