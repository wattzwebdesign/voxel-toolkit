AI Bot

The AI Bot function adds an intelligent search assistant to your Voxel site. Users can ask natural language questions like "find me a hotel near the beach" or "show apartments under $1000" and receive relevant results displayed as Voxel preview cards.

Overview

AI Bot transforms how visitors search your site. Instead of navigating complex filters, users simply describe what they're looking for in plain language. The AI interprets their request, searches your Voxel post types, and displays matching results in a sleek side panel.

Key Features

  • Natural language search across all Voxel post types
  • Results displayed as Voxel preview cards
  • Conversation memory for follow-up questions
  • Suggested query chips for quick searches
  • Configurable panel position (left/right)
  • Push or overlay panel behavior
  • Rate limiting to control API usage
  • Access control (everyone or logged-in only)
  • Full color and style customization
  • Works with Voxel's Action (VX) widget

Requirements

Before enabling AI Bot, you must configure AI settings:

  1. Navigate to Voxel Toolkit > AI Settings
  2. Choose your AI provider (OpenAI or Anthropic)
  3. Enter your API key
  4. Select your preferred model
  5. Click Save Changes

Note: AI Bot uses your configured AI provider for processing queries. Each user question consumes API tokens based on your provider's pricing.

Setup Instructions

Step 1: Enable the Function

  1. Navigate to Voxel Toolkit > Functions
  2. Find AI Bot in the list
  3. Toggle the switch to enable
  4. Click Save Changes

Step 2: Configure Post Types

  1. Click the settings icon next to AI Bot
  2. Under Post Types, select which post types the AI can search
  3. For each post type, optionally select a Card Template to display results
  4. Click Save Changes

Step 3: Add Trigger Button

The AI Bot panel is triggered using Voxel's Action (VX) widget:

  1. Edit any page in Elementor
  2. Add the Action (VX) widget where you want the trigger button
  3. In widget settings, find Action dropdown
  4. Select Open AI Assistant
  5. Style the button as desired
  6. Save the page

You can place multiple trigger buttons anywhere on your site - in the header, sidebar, floating buttons, or within page content.

Style Customization

Customize the AI Bot panel to match your site's design:

Colors

  • Primary Color - Header background and send button
  • Header Text Color - Text color in the header
  • AI Bubble Color - Background of AI message bubbles
  • AI Text Color - Text color in AI messages
  • User Bubble Color - Background of user message bubbles
  • User Text Color - Text color in user messages

Sizes

  • Panel Width - Width in pixels (default: 400)
  • Font Size - Base font size (default: 14px)
  • Border Radius - Roundness of message bubbles (default: 18px)

Avatar

Upload a custom avatar image for the AI assistant to display in chat bubbles.

Suggested Queries

Add suggested queries to help users understand what they can ask. These appear as clickable chips below the welcome message.

Example suggestions:

  • "Find hotels near the beach"
  • "Show me apartments under $1500"
  • "Pet-friendly rentals"
  • "Restaurants open now"

To add suggestions, enter one per line in the Suggested Queries field in settings.

Custom System Prompt

The system prompt tells the AI how to behave and what your site is about. A default prompt is generated automatically based on your post types and fields, but you can customize it for better results.

Tips for custom prompts:

  • Describe what your site offers
  • Mention any specific terminology users might use
  • Set the AI's tone (friendly, professional, etc.)
  • Specify any limitations or focus areas

How It Works

  1. User clicks the trigger button (Action VX widget)
  2. AI Bot panel slides open
  3. User types a natural language question
  4. AI analyzes the question and generates search parameters
  5. Voxel's search system finds matching posts
  6. Results are displayed as preview cards in the panel
  7. User can click cards to view full listings
  8. User can ask follow-up questions to refine results

Troubleshooting

AI Bot Not Appearing

  • Verify AI Settings are configured with a valid API key
  • Check that AI Bot function is enabled
  • Ensure you've added an Action (VX) widget with "Open AI Assistant" action
  • Clear any caching plugins

No Results Returned

  • Make sure at least one post type is selected in settings
  • Verify posts exist in the selected post types
  • Check that posts are published (not draft)
  • Try simpler queries to test

Slow Response Times

  • AI responses depend on your provider's server speed
  • Consider using a faster model if available
  • Reduce max results if searching returns too many posts

Rate Limit Errors

  • Increase the rate limit settings
  • Or disable rate limiting if you trust your users
  • Consider restricting to logged-in users only

Cards Not Displaying Correctly

  • Select a specific card template for each post type
  • Ensure the template is properly configured in Voxel
  • Check for CSS conflicts with your theme

API Usage & Costs

Each user question sends a request to your AI provider (OpenAI or Anthropic). Costs depend on:

  • Number of user queries
  • Length of conversation history (if memory enabled)
  • Size of your post type schema
  • Your chosen AI model

To control costs:

  • Enable rate limiting
  • Restrict to logged-in users
  • Use a less expensive AI model
  • Reduce conversation memory length
  • Monitor usage in your AI provider dashboard

Requirements
