SMS Providers Update - Solapi & Telnyx

Version 1.6.1 adds two new SMS providers to the Voxel Toolkit SMS integration: Solapi for the Korean market and Telnyx for international messaging. Both providers integrate seamlessly with Voxel's notification system.

Available SMS Providers

Voxel Toolkit now supports five SMS providers:

  • Twilio - Popular global SMS provider
  • Vonage (Nexmo) - International messaging platform
  • MessageBird - European SMS provider
  • Telnyx - Global telecommunications (NEW)
  • Solapi - Korean SMS provider (NEW)

Telnyx Provider

Telnyx is a global telecommunications provider offering competitive SMS rates and reliable delivery worldwide.

Features

  • Global SMS coverage
  • Competitive pricing
  • High delivery rates
  • Simple API key authentication
  • Real-time delivery status

Setup

  1. Create a Telnyx account at telnyx.com
  2. Purchase a phone number in the Mission Control Portal
  3. Generate an API key under API Keys section
  4. In Voxel Toolkit SMS settings, select Telnyx as provider
  5. Enter your API Key and From Number
  6. Save and test the configuration

Solapi Provider

Solapi is a Korean SMS provider optimized for the Korean market with local phone number support and compliance with Korean telecommunications regulations.

Features

  • Optimized for Korean market
  • Local phone number support
  • Korean language SMS
  • HMAC-SHA256 authentication
  • Regulatory compliance

Setup

  1. Create a Solapi account at solapi.com
  2. Verify your account and business information
  3. Register a sender phone number
  4. Generate API credentials in the console
  5. In Voxel Toolkit SMS settings, select Solapi as provider
  6. Enter your API Key, API Secret, and From Number
  7. Save and test the configuration

Required Credentials

Phone Number Format

Solapi uses Korean phone number format. The provider automatically handles format conversion:

  • International format (+821012345678) is converted to local format
  • Local format (01012345678) is used as-is

Configuration

Accessing SMS Settings

  1. Navigate to Voxel Toolkit > Functions
  2. Find the SMS Notifications function
  3. Click the settings icon
  4. Select your preferred provider from the dropdown
  5. Enter the required credentials
  6. Save changes

Testing Your Configuration

After configuring a provider, use the test feature to verify your setup:

  1. Enter a test phone number
  2. Click "Send Test SMS"
  3. Check for delivery confirmation
  4. Verify the message was received on the test phone

Troubleshooting

Telnyx Issues

Authentication Failed:

  • Verify API key is correct and active
  • Check that the key has SMS permissions
  • Ensure you're using a v2 API key

Messages Not Delivering:

  • Verify phone number is in E.164 format
  • Check account balance
  • Confirm sender number is approved for SMS

Solapi Issues

Authentication Failed:

  • Verify both API Key and API Secret are correct
  • Check server time is synchronized (HMAC is time-sensitive)
  • Ensure credentials have not expired

Korean Number Required:

  • Solapi requires a registered Korean sender number
  • Verify sender number is approved in Solapi console
  • Check Korean telecommunications registration status

Requirements

  • Voxel Theme (active)
  • Voxel Toolkit plugin (enabled)
  • SMS Notifications function enabled
  • Active account with chosen provider
  • Valid API credentials
  • Approved sender phone number
