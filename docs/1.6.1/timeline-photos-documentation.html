Timeline Photos Widget (Updated for 1.6.1)

The Timeline Photos Widget displays photos from timeline entries in a beautiful gallery format. Version 1.6.1 adds new source options to pull photos from different timeline types.

What's New in 1.6.1

New source options allow you to display photos from different timeline types:

  • Post Wall: Photos from post wall entries
  • Post Timeline: Photos from post timeline entries
  • Author Timeline: Photos from the author's user timeline

Overview

This Elementor widget collects and displays photos from timeline entries associated with a post or author. Perfect for showcasing user-generated content, review photos, or activity feeds in a visual gallery format.

Key Features

  • Collect photos from multiple timeline sources
  • Grid gallery display
  • Configurable number of photos
  • Lightbox support
  • Full styling controls
  • Hide if empty option

Setup Instructions

Step 1: Enable the Widget

  1. Navigate to Voxel Toolkit > Widgets
  2. Find Timeline Photos in the list
  3. Toggle the switch to enable
  4. Click Save Changes

Step 2: Add Widget to Template

  1. Edit your post type template in Elementor
  2. Search for Timeline Photos (VT)
  3. Drag the widget to your desired location
  4. Configure source and display options
  5. Save the template

Widget Settings

Content Settings

Styling Options

  • Gap: Spacing between photos
  • Border Radius: Roundness of photo corners
  • Aspect Ratio: Photo aspect ratio (square, 4:3, 16:9, etc.)
  • Object Fit: How images fill the container (cover, contain)
  • Hover Effect: Opacity, scale, or overlay on hover

Use Cases

Review Photo Gallery

Display customer review photos on product or service pages using the Post Reviews source.

Community Wall Gallery

Show user-submitted photos from the community wall using the Post Wall source.

Activity Photo Stream

Display recent photos from post updates using the Post Timeline source.

Author Portfolio

Show photos from a user's timeline on their profile using the Author Timeline source.

Troubleshooting

No Photos Displaying

  • Verify the selected source has photos
  • Check that "Hide if Empty" isn't enabled
  • Ensure timeline entries contain image attachments
  • Verify the post has the selected timeline type

Wrong Photos Showing

  • Check the source setting matches your intent
  • Each source pulls from different timeline types
  • Author Timeline shows user's photos, not post photos

Lightbox Not Working

  • Ensure "Enable Lightbox" is turned on
  • Check for JavaScript errors
  • Verify lightbox library is loading

Requirements

  • Voxel Theme (active)
  • Voxel Toolkit plugin (enabled)
  • Elementor (for widget)
  • Timeline entries with photos
