Voxel Toolkit 1.6.1.1 Release Notes

Version 1.6.1.1 is a bug fix release addressing issues discovered in 1.6.1.

Bug Fixes

Messenger Widget - Persistent Admin Chat Fix

Fixed an issue where the persistent admin chat circle would duplicate exponentially (doubling on each render). The fix ensures only one instance of the persistent admin chat and AI bot circles exist by:

  • Using .first() on container and popup selections to work with single widget instances
  • Removing duplicate circles before re-rendering the chat list
  • Adding duplicate checks before prepending circles back to the list

External Link Warning - Settings Not Saving

Fixed an issue where border colors and button styling settings were not being saved in the External Link Warning feature. Added missing sanitization for:

  • Continue button border color
  • Cancel button border color
  • Button border radius
  • Button border width
