Poll Display Widget

The Poll Display widget shows interactive polls from Voxel poll fields. Users can vote on options, see real-time results with percentages, and optionally add their own options to the poll.

Overview

Display polls created with the Voxel Toolkit Poll Field. The widget shows all poll options with voting functionality, progress bars showing results, and the ability for users to submit their own options if enabled.

Key Features

  • Radio or checkbox voting (single/multiple choice)
  • Real-time vote counting and percentages
  • Visual progress bars for each option
  • User-submitted options (optional)
  • Customizable text labels
  • Full styling controls
  • Vote change support

Prerequisites

Before using this widget, you need to:

  1. Enable the Poll Field in Voxel Toolkit > Post Fields
  2. Add a poll field to your post type in Voxel > Post Types
  3. Configure poll options in the field settings

Setup Instructions

Step 1: Enable the Widget

  1. Navigate to Voxel Toolkit > Widgets in your WordPress admin
  2. Find Poll Display in the list
  3. Toggle the switch to enable the widget
  4. Click Save Changes

Step 2: Add to Template

  1. Edit your single post template in Elementor
  2. Search for Poll Display (VT)
  3. Drag the widget to your desired location
  4. Select the poll field to display
  5. Save the template

Poll Types

Single Choice (Radio)

Users can only select one option. When they vote for a different option, their previous vote is removed.

Multiple Choice (Checkbox)

Users can select multiple options. Each checkbox can be toggled independently.

The poll type is configured in the poll field settings, not in the widget.

User-Submitted Options

When enabled in the poll field:

  • Logged-in users can add their own options
  • User-submitted options show who submitted them
  • New options appear immediately and can be voted on

Troubleshooting

"Please select a poll field"

  • Ensure the Poll Field is enabled in Toolkit settings
  • Add a poll field to your post type
  • Select the field in the widget settings

"No poll data available"

  • Add options to the poll field in the post
  • Check that the post has poll data saved

Voting Not Working

  • Users must be logged in to vote
  • Check browser console for JavaScript errors
  • Verify AJAX requests aren't blocked

Requirements

  • Voxel Theme (active)
  • Voxel Toolkit plugin (enabled)
  • Poll Field enabled
  • Elementor (for widget placement)
  • Users must be logged in to vote
