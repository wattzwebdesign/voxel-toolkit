Reading Time Widget

The Reading Time widget displays an estimated reading time for posts based on word count. Help visitors know how long it will take to read an article before they start.

Overview

Calculate and display how long it takes to read a post. The widget counts words in the content and calculates reading time based on an average reading speed. Available as both an Elementor widget and a shortcode.

Key Features

  • Automatic word count calculation
  • Customizable reading speed (words per minute)
  • Prefix and postfix text options
  • Alignment controls
  • Typography and color styling
  • Elementor widget and shortcode support

Setup Instructions

Step 1: Enable the Widget

  1. Navigate to Voxel Toolkit > Widgets in your WordPress admin
  2. Find Reading Time in the list
  3. Toggle the switch to enable the widget
  4. Click Save Changes

Step 2: Add to Template

  1. Edit your single post template in Elementor
  2. Search for Reading Time
  3. Drag the widget below the post title or in the meta area
  4. Configure text and styling
  5. Save the template

Shortcode Usage

Use the shortcode for non-Elementor pages:

[voxel_reading_time prefix="⏱️ " postfix=" minute read"]

How Reading Time is Calculated

  1. The post content is retrieved
  2. Shortcodes and HTML are stripped
  3. Words are counted
  4. Word count is divided by words per minute
  5. Result is rounded up (minimum 1 minute)

Reading Speed Guidelines

  • 200 WPM - Slower, more careful reading
  • 250 WPM - Average reading speed
  • 300 WPM - Average (default)
  • 350+ WPM - Fast readers

Styling Options

In Elementor, you can style:

  • Prefix color and typography
  • Reading time value color and typography
  • Postfix color and typography
  • Container alignment

Troubleshooting

Shows "1 min" for All Posts

  • Minimum reading time is 1 minute
  • Very short posts will show 1 minute

Widget Not Showing

  • Check that the widget is enabled in Toolkit settings
  • Verify you're viewing a single post (not an archive)
  • Ensure the post has content

Inaccurate Reading Time

  • Adjust words per minute based on your audience
  • Technical content may need lower WPM (slower reading)
  • Light content may use higher WPM (faster reading)

Requirements

  • Voxel Theme (active)
  • Voxel Toolkit plugin (enabled)
  • Elementor (for widget placement) or shortcode support
