Onboarding Widget

The Onboarding widget creates interactive guided tours to help users navigate your website. Walk visitors through your site's features with step-by-step tooltips that highlight specific elements on the page.

Overview

Create engaging onboarding experiences with interactive tours. The widget uses intro.js to display tooltips that guide users through your site, highlighting specific elements and explaining features. Tours can auto-start or be triggered by a button.

Key Features

  • Multi-step guided tours
  • Target any element by CSS selector
  • Auto-start or button-triggered tours
  • Progress bar and step bullets
  • Tour versioning for updates
  • Remembers completion status
  • Full styling customization
  • Preview mode in editor

Setup Instructions

Step 1: Enable the Widget

  1. Navigate to Voxel Toolkit > Widgets in your WordPress admin
  2. Find Onboarding in the list
  3. Toggle the switch to enable the widget
  4. Click Save Changes

Step 2: Add to Page

  1. Edit your page in Elementor
  2. Search for Onboarding (VT)
  3. Drag the widget to your page
  4. Configure tour steps and settings
  5. Save the page

Step 3: Create Tour Steps

  1. Add a step with a title and content
  2. Enter the target element's CSS selector
  3. Choose tooltip position
  4. Repeat for additional steps

Targeting Elements

To target specific elements on your page:

Using CSS IDs

  1. Select any Elementor widget
  2. Go to Advanced > CSS ID
  3. Enter an ID (e.g., "search-box")
  4. In the tour step, enter #search-box

Using CSS Classes

Target existing classes with .class-name

Full-Page Overlay

Leave the target selector empty for a centered tooltip with full-page overlay.

Tour Versioning

When you update your tour and want all users to see it again:

  1. Click the Reset Tour for All Users button in the widget settings
  2. This increments the tour version
  3. Users who completed the old version will see the new tour

Troubleshooting

Tour Not Appearing

  • If auto-start is enabled, tour only shows once per session
  • Clear browser storage to reset completion status
  • Use the Reset Tour button to make it appear again

Element Not Highlighting

  • Verify the CSS selector is correct
  • IDs should start with # (e.g., #my-element)
  • Classes should start with . (e.g., .my-class)
  • Make sure the element exists on the page

Tooltip in Wrong Position

  • Try a different tooltip position setting
  • Use "auto" to let it position automatically

Preview Tour

Use the Preview Tour button in the Elementor editor to test your tour without affecting the auto-start behavior or completion status.
