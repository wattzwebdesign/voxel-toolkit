Timeline Photos Widget

The Timeline Photos widget displays photos that have been uploaded with reviews for a post. Create a beautiful gallery of user-submitted review photos from Voxel's timeline system.

Overview

Collect and display photos from reviews in a responsive gallery layout. When users submit reviews with photos, those images are automatically available to display on the listing page. Show off real customer photos alongside your listing content.

Key Features

  • Grid or masonry layout options
  • Lightbox support for full-size viewing
  • Configurable number of columns
  • Multiple image size options
  • Empty state message customization
  • Elementor widget and shortcode support
  • Debug mode for troubleshooting

Setup Instructions

Step 1: Enable the Widget

  1. Navigate to Voxel Toolkit > Widgets in your WordPress admin
  2. Find Timeline Photos in the list
  3. Toggle the switch to enable the widget
  4. Click Save Changes

Step 2: Add to Template

  1. Edit your single post template in Elementor
  2. Search for Timeline Photos
  3. Drag the widget to your desired location
  4. Configure layout settings
  5. Save the template

How It Works

The widget queries Voxel's timeline database for review entries associated with the post. It extracts photo attachment IDs from the review details and displays them in a responsive gallery.

  1. Widget queries the voxel_timeline table for post reviews
  2. Extracts file IDs from the review details JSON
  3. Fetches image attachment data for each file ID
  4. Renders the gallery with configured layout

Troubleshooting

No Photos Appearing

  • Verify reviews with photos exist for the post
  • Check that the voxel_timeline table exists in the database
  • Use debug mode to see timeline data
  • Ensure photos were uploaded as attachments

Using Debug Mode

Add debug="yes" to the shortcode to see detailed information about timeline entries:

[timeline_photos debug="yes"]

Photos Not Displaying Correctly

  • Check that the image size exists in WordPress
  • Verify attachment files haven't been deleted
  • Regenerate thumbnails if needed

Requirements

  • Voxel Theme (active)
  • Voxel Toolkit plugin (enabled)
  • Elementor (for widget placement) or shortcode support
  • Voxel review system with file uploads enabled
  • Reviews with photos in the timeline database
