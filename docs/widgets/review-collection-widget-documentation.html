Review Collection Widget

The Review Collection widget displays a grid of reviews from Voxel's review system. Showcase positive reviews from across your site, filter by post type, and customize the layout to match your design.

Overview

Pull reviews from Voxel's timeline and display them in a beautiful grid layout. Filter reviews by minimum rating, post type, or specific posts. Each review card shows the reviewer, rating stars, review content, and the reviewed post.

Key Features

  • Responsive grid layout
  • Minimum rating filter
  • Post type filtering
  • Content line limiting
  • Customizable star ratings
  • Author avatars and names
  • Full styling controls

Setup Instructions

Step 1: Enable the Widget

  1. Navigate to Voxel Toolkit > Widgets in your WordPress admin
  2. Find Review Collection in the list
  3. Toggle the switch to enable the widget
  4. Click Save Changes

Step 2: Add to Page

  1. Edit your page in Elementor
  2. Search for Review Collection (VT)
  3. Drag the widget to your layout
  4. Configure content and styling
  5. Save the page

Filtering Reviews

By Minimum Rating

Set a minimum rating (1-5 stars) to only show positive reviews. Most sites use 4 or 5 stars to display only the best reviews.

By Post Type

Enter post type keys separated by commas:

places,events,profile

For Specific Post

Enter a post ID to show only reviews for that listing/post.

From Specific Author

Enter a user ID to show only reviews written by that person.

Use Cases

Homepage Testimonials

Display top reviews across all post types to showcase social proof.

Post Type Archives

Show reviews specific to a category or post type.

Listing Pages

Display reviews for a specific listing using the Post ID filter.

User Profiles

Show all reviews written by a specific user using the Author ID filter.

Troubleshooting

No Reviews Showing

  • Check that reviews exist in the Voxel timeline
  • Lower the minimum rating filter
  • Disable "Only With Message" to include all reviews
  • Check post type filter is correct

Wrong Reviews Appearing

  • Check your post type filter settings
  • Verify the post ID or author ID filters

Layout Issues

  • Adjust column count for different screen sizes
  • Check gap and padding settings
  • Adjust content line limit if text is cut off

Requirements

  • Voxel Theme (active)
  • Voxel Toolkit plugin (enabled)
  • Elementor (for widget placement)
  • Voxel review/rating system active
  • Reviews in the Voxel timeline database
