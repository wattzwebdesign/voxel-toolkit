Profile Progress Widget

The Profile Progress widget displays how complete a user's profile is. Show a progress bar or circular indicator based on which profile fields have been filled out, encouraging users to complete their profiles.

Overview

Track profile completion by checking which fields users have filled out. Display progress as a horizontal bar or circular indicator with an optional percentage and field checklist. Available as both an Elementor widget and a shortcode.

Key Features

  • Horizontal or circular progress display
  • Configurable field selection
  • Percentage display option
  • Optional field checklist
  • Works with all Voxel profile fields
  • Elementor widget and shortcode support

Setup Instructions

Step 1: Enable the Widget

  1. Navigate to Voxel Toolkit > Widgets in your WordPress admin
  2. Find Profile Progress in the list
  3. Toggle the switch to enable the widget
  4. Click Save Changes

Step 2: Add to Template

  1. Edit your profile or dashboard template in Elementor
  2. Search for Profile Progress
  3. Drag the widget to your desired location
  4. Select which fields to track
  5. Save the template

Configuration Options

Progress Types

Horizontal Bar

A standard progress bar that fills from left to right based on completion percentage.

Circular Progress

A circular/ring indicator that fills based on completion percentage. Great for dashboard displays.

Shortcode Usage

Use the shortcode for non-Elementor pages:

[voxel_profile_progress field_keys="name,bio,avatar" progress_type="horizontal" show_percentage="yes"]

How Progress is Calculated

The widget checks each selected field to see if it has a value:

  • Fields with any content are marked as complete
  • Empty fields are marked as incomplete
  • Percentage = (completed fields / total fields) Ã— 100

Troubleshooting

Progress Shows 0%

  • Verify user has a Voxel profile created
  • Check that selected field keys are correct
  • Ensure user is logged in (if checking current user)

"No profile fields found"

  • Verify Voxel's profile post type is configured
  • Check that profile fields exist in Voxel settings

Widget Not Appearing

  • Widget only shows for logged-in users by default
  • Check that the widget is enabled in Toolkit settings

Requirements

  • Voxel Theme (active)
  • Voxel Toolkit plugin (enabled)
  • Elementor (for widget placement) or shortcode support
  • Voxel profile post type configured
