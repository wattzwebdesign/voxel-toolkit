Campaign Progress Widget

The Campaign Progress widget displays donation and crowdfunding progress for fundraising campaigns. Show your visitors how much has been raised, the goal amount, and a visual progress bar with recent donor information.

Overview

Track and display fundraising progress for campaigns, donations, or crowdfunding goals. The widget shows the amount raised, remaining amount, percentage progress, donation count, and optionally a list of recent donors with their contributions.

Key Features

  • Visual progress bar showing fundraising status
  • Customizable goal amount and currency
  • Display total raised vs. goal
  • Show donation count
  • Recent donor list with avatars
  • Customizable text labels
  • Full styling controls

Setup Instructions

Step 1: Enable the Widget

  1. Navigate to Voxel Toolkit > Widgets in your WordPress admin
  2. Find Campaign Progress in the list
  3. Toggle the switch to enable the widget
  4. Click Save Changes

Step 2: Add to Template

  1. Edit your campaign or fundraising post template in Elementor
  2. Search for Campaign Progress
  3. Drag the widget to your desired location
  4. Configure goal amount and display options
  5. Save the template

How It Works

The widget displays:

  1. Progress Summary - Shows remaining amount and goal (e.g., "£200 left of Goal £510")
  2. Progress Bar - Visual bar showing percentage filled with amount raised and percentage
  3. Donation Count - Total number of donations received
  4. Donor List - Recent donors with avatars, names, dates, and amounts

Troubleshooting

Progress Bar Shows 0%

  • Verify donations have been made to this campaign
  • Check that the goal amount is set correctly

Donor List Not Appearing

  • Ensure "Display Donor List" is enabled
  • Check that donations exist for this campaign

Wrong Currency Symbol

  • Update the Currency Symbol field in widget settings
  • Common symbols: $ (USD), £ (GBP), € (EUR), ¥ (JPY)

Requirements

  • Voxel Theme (active)
  • Voxel Toolkit plugin (enabled)
  • Elementor (for widget placement)
  • Voxel product/donation system configured
