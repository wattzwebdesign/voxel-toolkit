Pre-Approve Posts

The Pre-Approve Posts function automatically publishes posts from trusted users without requiring manual review. Configure approval based on user roles, verified profile status, or manual user selection.

Overview

By default, Voxel posts go to "pending" status for admin review. This function allows trusted users to bypass the review queue and have their posts published immediately.

Key Features

  • Automatic publishing for pre-approved users
  • Role-based approval
  • Verified profile approval
  • Manual per-user approval
  • User list column showing approval status
  • Works with all Voxel post types

Setup Instructions

Step 1: Enable the Function

  1. Navigate to Voxel Toolkit > Functions in your WordPress admin
  2. Find Pre-Approve Posts in the list
  3. Toggle the switch to enable the function
  4. Click Save Changes

Step 2: Configure Approval Methods

Choose how users become pre-approved:

By User Role

  1. In the Approved Roles section, select which roles are pre-approved
  2. All users with the selected roles will have their posts auto-published
  3. Click Save Changes

By Verified Status

  1. Enable the Approve Verified Users option
  2. Users with verified Voxel profiles will be pre-approved
  3. Click Save Changes

By Manual Selection

  1. Go to Users in WordPress admin
  2. Edit any user's profile
  3. Find Voxel Toolkit Settings section
  4. Check Pre-Approved for Publishing
  5. Save the user profile

Approval Methods Priority

Users are considered pre-approved if ANY of the following are true:

Users List Column

A "Pre-Approved?" column is added to the Users list showing:

  • ✓ Manual - User is manually pre-approved
  • ✓ Verified - User has verified profile
  • ✓ Role - User has an approved role
  • Empty - User is not pre-approved

You can disable this column in the function settings if desired.

How It Works

When a pre-approved user submits a post:

  1. Post is created with "pending" status (normal Voxel behavior)
  2. Function detects the new post
  3. Checks if author is pre-approved
  4. If approved, status is immediately changed to "publish"
  5. Post is live without admin intervention

Use Cases

Trusted Vendors

Allow established vendors to publish listings instantly while new users require review.

Premium Members

Grant pre-approval to users with premium membership roles as a perk.

Verified Businesses

Use Voxel's profile verification to establish trust and auto-approve verified businesses.

Combining with Custom Submission Messages

If you use the Custom Submission Messages function, you can configure different messages for pre-approved users:

  • Pending Review: "Your listing has been submitted for review."
  • Pre-Approved: "Your listing is now live!"

Troubleshooting

Posts Still Going to Pending

  • Verify the function is enabled
  • Check that the user is actually pre-approved (check Users list column)
  • Ensure no other plugins are modifying post status

User Not Showing as Pre-Approved

  • For role approval: verify user has the approved role
  • For verified approval: check user's profile has voxel:verified meta
  • For manual approval: check user profile settings

Column Not Appearing in Users List

  • Ensure "Show Column" option is enabled in settings
  • Check Screen Options in Users list to ensure column is visible

Requirements

  • Voxel Theme (active)
  • Voxel Toolkit plugin (enabled)
  • Voxel post types configured
