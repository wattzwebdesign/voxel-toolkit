Verified Reviews

The Verified Reviews function ensures review authenticity by requiring purchase verification and adding "Purchased" badges to verified buyer reviews. Control who can leave reviews and who can respond as a business owner.

Overview

Build trust with authentic reviews. This function restricts review submissions to verified purchasers and adds visual badges to indicate which reviewers have actually bought from a listing. Business owners need active memberships or listings to respond to reviews.

Key Features

  • Restrict reviews to verified purchasers
  • "Purchased" badge on verified buyer reviews
  • Per-post-type purchase requirements
  • Author response restrictions
  • Requires active membership or listing to respond
  • Customizable badge label and color
  • Customizable error messages

Setup Instructions

Step 1: Enable the Function

  1. Navigate to Voxel Toolkit > Functions in your WordPress admin
  2. Find Verified Reviews in the list
  3. Toggle the switch to enable the function
  4. Click Save Changes

Step 2: Configure Purchase Requirements

  1. Select which post types require purchase to review
  2. Configure error messages for non-purchasers
  3. Click Save Changes

Step 3: Customize Badge Appearance

  1. Set the badge label (default: "Purchased")
  2. Choose badge color
  3. Select badge icon
  4. Click Save Changes

How It Works

Review Submission

  1. User tries to submit a review
  2. Function checks if post type requires purchase
  3. If required, checks user's order history
  4. Review allowed only if purchase found
  5. Non-purchasers see error message

Badge Display

  1. Review is displayed on listing page
  2. Function checks if reviewer has purchased
  3. If verified, badge is added to review
  4. Badge appears next to reviewer's name

Author Response Permission

Listing owners can respond to reviews only if they have:

  • Active paid membership (not the default/free plan), OR
  • Active paid listing (completed or active subscription order)

Use Cases

E-commerce Listings

Ensure product reviews come only from verified buyers, eliminating fake reviews.

Service Marketplaces

Only allow reviews from customers who have actually booked and paid for services.

Trust Building

The "Purchased" badge shows visitors which reviews are from real customers.

Premium Response Feature

Encourage businesses to upgrade to paid plans to respond to customer reviews.

Troubleshooting

Badge Not Appearing

  • Verify the function is enabled
  • Check that reviewer has a completed order for that listing
  • Ensure CSS is loaded (check for verified-reviews.css)
  • Clear any caching

Users Can't Leave Reviews

  • Check if post type is in the "purchase required" list
  • Verify user has a completed order
  • Order must be for the specific listing being reviewed

Owner Can't Respond

  • Owner needs active paid membership OR active paid listing
  • Default/free membership doesn't qualify
  • Check order status is "completed" or "sub_active"

Requirements

  • Voxel Theme (active)
  • Voxel Toolkit plugin (enabled)
  • Voxel orders/payments enabled
  • Stripe integration (for purchase verification)
