Field Formatters

The Field Formatters function provides shared utility methods for formatting Voxel field values for display. Convert complex field data like work hours and locations into human-readable formats.

Overview

Voxel stores complex field data in arrays and JSON formats. This utility function provides methods to format that data into readable strings for display in templates, widgets, and custom integrations.

Key Features

  • Format work hours schedules
  • Format location/address data
  • Day name translations
  • Status handling (open, closed, by appointment)
  • Multiple time slot support
  • Safe output escaping

Setup Instructions

Enable the Function

  1. Navigate to Voxel Toolkit > Functions in your WordPress admin
  2. Find Field Formatters in the list
  3. Toggle the switch to enable the function
  4. Click Save Changes

Available Formatters

Work Hours Formatter

Converts Voxel work hours field data into readable schedule display.

Example Output

Mon, Tue, Wed: 9:00 - 17:00
Thu, Fri: 9:00 - 12:00, 13:00 - 17:00
Sat: 10:00 - 14:00
Sun: Closed

Location Formatter

Extracts and formats address from Voxel location field data.

Handles:

  • JSON encoded location data
  • Array with address key
  • Plain string addresses

Use Cases

Custom Templates

Display formatted work hours in custom Elementor templates or widgets.

Include readable schedule information in automated emails.

Format field data for export to spreadsheets.

Admin Displays

Show formatted field values in admin columns or meta boxes.

Troubleshooting

"No hours set" Showing

  • Verify the work hours field has data
  • Check the schedule array structure
  • Ensure at least one day group is configured

"No location set" Showing

  • Verify location field has an address
  • Check if data is properly JSON encoded
  • Ensure 'address' key exists in location data

Requirements

  • Voxel Theme (active)
  • Voxel Toolkit plugin (enabled)
