VERSION 1.5.7

Voxel Toolkit Update

New features & bug fixes since 1.5.6

+

New Features

  • Calendar Week Start
    Date pickers now respect your WordPress "Week Starts On" setting. No more Monday-only calendars!
  • Share Count Tracking
    Track share button clicks with dynamic tags. Use @post(share_count) for total or @post(share_count.facebook) for per-network stats.
  • Auto Reply Dynamic Tag
    Auto Reply field values now available as dynamic tags for use in your templates.
  • Table of Contents - Field Indicators
    Show individual fields under each step with completion status. Empty/filled circle indicators update in real-time.
  • Docs Menu Link
    Quick access to documentation right from your WordPress admin menu.
✓

Bug Fixes

  • Messenger Settings on Cloudways
    Fixed "The link you followed has expired" error on cached hosting environments.
  • Messenger Widget - Reply As
    Added translatable "Reply As" text setting. Fixed scrollbar on long listing names.
  • Membership Plan Filter
    Fixed filter not working on sites using Voxel's Stripe test mode.
  • SMS Notifications Toggle
    Fixed 400 error when toggling SMS in App Events page.
  • AI Review Summary
    Fixed OpenAI API key not saving in Toolkit Settings.
  • Suggest Edits Widget
    Fixed taxonomy terms showing IDs instead of labels. Added option to hide "Permanently Closed".
