Admin Taxonomy Search

The Admin Taxonomy Search function adds a search box to taxonomy metaboxes in the post edit screen. Quickly find and select terms when you have large taxonomies.

Overview

When taxonomies have many terms, scrolling through the list becomes tedious. This function adds a search/filter input that instantly filters the term list as you type.

Key Features

  • Search box in taxonomy metaboxes
  • Instant filtering as you type
  • Configurable per taxonomy
  • Works on post and page edit screens
  • Shows "no results" message when no matches

Setup Instructions

Step 1: Enable the Function

  1. Navigate to Voxel Toolkit > Functions in your WordPress admin
  2. Find Admin Taxonomy Search in the list
  3. Toggle the switch to enable the function
  4. Click Save Changes

Step 2: Select Taxonomies

  1. In the Taxonomies section, check which taxonomies should have the search feature
  2. Common selections: Categories, Tags, and Voxel custom taxonomies
  3. Click Save Changes

How to Use

  1. Open any post or page for editing
  2. Find the taxonomy metabox in the sidebar (e.g., Categories)
  3. Look for the search input at the top of the term list
  4. Start typing to filter terms
  5. Terms matching your search will be shown
  6. Click to select/deselect terms as normal

Use Cases

Large Category Lists

When you have hundreds of categories, quickly find the right one without scrolling.

Location Taxonomies

For Voxel sites with location-based taxonomies (cities, regions), search makes selection much faster.

Product Attributes

WooCommerce or custom product taxonomies with many options become manageable.

Troubleshooting

Search Box Not Appearing

  • Ensure the function is enabled
  • Verify the specific taxonomy is selected in settings
  • Check that you're on a post edit screen (not the taxonomy management page)
  • Clear browser cache

Search Not Filtering

  • Check browser console for JavaScript errors
  • Ensure the JavaScript file is loading correctly
  • Try disabling other plugins that modify the post edit screen

Terms Not Being Found

  • Search matches term names only (not descriptions)
  • Search is case-insensitive
  • Check spelling of your search term

Requirements

  • Voxel Theme (active)
  • Voxel Toolkit plugin (enabled)
  • Taxonomy must display as checkbox list in metabox
