Light Mode (Admin)

The Light Mode function transforms Voxel's dark admin interface into a clean, light-colored theme. This provides better readability in bright environments and matches the standard WordPress admin aesthetic.

Overview

Voxel's admin pages use a dark color scheme by default. This function applies a comprehensive light theme to all Voxel admin interfaces, making them easier to read and more consistent with the standard WordPress admin appearance.

Key Features

  • Light background for all Voxel admin pages
  • Improved text contrast and readability
  • Styled form inputs, buttons, and controls
  • Modal and popup theming
  • Excludes Voxel Toolkit's own pages (maintains separate styling)

Setup Instructions

Enable the Function

  1. Navigate to Voxel Toolkit > Functions in your WordPress admin
  2. Find Light Mode in the list
  3. Toggle the switch to enable the function
  4. Click Save Changes

The light theme will immediately apply to all Voxel admin pages.

Affected Pages

Light mode applies to the following Voxel admin pages:

  • Voxel Settings pages (pages with "voxel" in the URL)
  • Post Type Editor (edit-post-type-* pages)
  • All Voxel configuration modals and popups

Excluded Pages

Light mode does NOT apply to:

  • Voxel Toolkit main page
  • Voxel Toolkit Settings page
  • Voxel Toolkit License Management page

Use Cases

Bright Environments

If you work in a well-lit office or during daytime, light mode reduces eye strain compared to dark interfaces.

Visual Consistency

Match the Voxel admin interface with the standard WordPress admin for a more consistent experience.

Client Training

Clients familiar with WordPress may find the light interface more intuitive and easier to navigate.

Troubleshooting

Styles Not Applying

  • Clear browser cache and refresh the page
  • Ensure the function is enabled in Voxel Toolkit > Functions
  • Check that you're on a Voxel admin page (not the standard WordPress admin)

Some Elements Still Dark

  • Some third-party Voxel add-ons may have their own styles
  • Voxel theme updates may introduce new elements not yet styled
  • Check for CSS conflicts with other plugins

Text Hard to Read

If text contrast seems low:

  • Try disabling other admin color scheme plugins
  • Check your monitor's brightness and contrast settings
  • Report specific pages where text is hard to read

Requirements

  • Voxel Theme (active)
  • Voxel Toolkit plugin (enabled)
  • Access to Voxel admin pages
