We're thrilled to announce Voxel Toolkit 1.6.0 – our most feature-packed release to date. This update brings powerful AI capabilities, a complete RSVP system, route planning, post comparison tools, and dozens of quality-of-life improvements that will transform how you build with Voxel.

AI-Powered Features

Artificial intelligence takes center stage in 1.6.0 with three groundbreaking features:

AI Bot - Natural Language Search

Let your visitors search naturally. Instead of clicking through filters, users can simply ask questions like "Find Italian restaurants near me" or "Events this weekend." The AI Bot understands context, remembers conversations, and delivers relevant results instantly.

  • Suggested query chips to guide users
  • Quick filters after results (4+ Stars, Has Reviews)
  • Follow-up actions on cards (Directions, Call, View)
  • Full appearance customization with live preview
  • Optional Messenger widget integration

AI Post Summaries

Automatically generate summaries for your listings when they're published or updated. Configure custom prompts per post type, and access summaries anywhere using the @post(ai.summary) dynamic tag. A bulk generation tool handles existing posts that need summaries.

Timeline Reply Summaries (TL;DR)

Long discussion threads? The AI generates concise summaries of replies so users can quickly catch up. Works on reviews, walls, timelines, and user feeds. Supports both OpenAI and Anthropic (Claude) providers.

RSVP System

A complete event RSVP solution is now built into Voxel Toolkit:

  • RSVP Form Widget – Let users submit or cancel RSVPs
  • Attendee List Widget – Display RSVPs with admin actions
  • Guest Support – Allow non-registered users to RSVP
  • Approval Workflow – Pending, approved, and rejected statuses
  • Capacity Limits – Set max attendees with automatic closure
  • CSV Export – Download attendee data for your records
  • Custom Fields – Define different RSVP fields per post type
  • App Events – Trigger notifications when RSVPs are submitted, approved, or rejected

Route Planner Widget

Display interactive routes with turn-by-turn directions. Perfect for tour itineraries, delivery routes, or multi-stop experiences.

  • Data from repeater fields, post relations, or individual location fields
  • Google Maps, Mapbox, or OpenStreetMap support
  • Driving, walking, cycling, and transit modes
  • Drag-to-reorder waypoints with automatic recalculation
  • Route optimization (nearest neighbor algorithm)
  • Step-by-step directions panel with highlighting
  • Export to Google Maps, Apple Maps, or GPX file

Compare Posts

Help your users make better decisions with side-by-side comparisons:

  • Compare 2-4 posts of the same type
  • Compare Button widget for single post templates
  • Comparison Table widget for dedicated pages
  • Floating comparison bar (bottom or side)
  • Configure which fields appear with custom labels
  • Cross-tab sync via localStorage
  • Print-friendly comparison tables

Team Members

Post authors can now invite collaborators to help manage their listings:

  • Invite team members by email
  • Configurable invite expiration (default 7 days)
  • Accept/decline flow with email notifications
  • App Events for invitations and responses
  • Elementor visibility rule: "User is team member of current post"

Saved Search

Let users save their search filters and get notified when new posts match:

  • Save Search button integrates with Voxel's search form
  • Saved Search widget displays saved searches in a grid
  • Auto-apply remembers your last search on page return
  • Notifications when new posts match (in-app, email, SMS)
  • Configurable auto-delete expiration
  • Email batching prevents server overload

Recurring Events – Multiple Instances

Events with multiple upcoming dates now appear as separate cards in your archives – one per occurrence. Each card shows only its specific date, sorted soonest first. Use @post(recurring-date.upcoming.start) in your card template to display the occurrence date.

Social Proof (Beta)

Display toast notifications showing recent activity on your site – bookings, orders, signups, reviews, and more. Features include:

  • Real-time event capture
  • Auto-rotation with configurable timing
  • Activity Boost mode for slower periods
  • Customizable position, colors, and animations
  • Message templates with placeholders

More New Features

Message Moderation

A new admin page under Messages (VT) lets you view and moderate all direct messages. Filter by status, search content, and take bulk actions – perfect for community management.

Temporary Login Links

Generate secure one-click login URLs for developers or support staff. No password sharing required. Features include multi-use tokens, configurable expiration, redirect URLs, and full audit logging.

External Link Warning

Protect your users with a customizable warning modal when they click links to external sites. Whitelist trusted domains and exclude specific elements.

Enhanced TinyMCE Editor

Voxel's WP Editor field now includes an Add Media button, text color pickers, and character map – giving your users richer content creation tools.

Advanced Phone Input

Phone fields now feature country selector dropdowns with flags and dial codes. Set default countries and restrict options per field.

Add Category from Frontend

Let users add new taxonomy terms directly from the Create Post form. Optional approval workflow keeps things moderated.

Enhanced Post Relation Display

Customize how posts appear in relation field dropdowns using dynamic tag templates. Show "Restaurant Name - City" instead of just the title.

Timeline Filters

Add an "Unanswered" filter to timeline widgets to surface posts without replies – great for support forums and Q&A sections.

Field Column Width Picker

Quickly set field widths in the post type editor with a visual dropdown (100%, 75%, 66%, 50%, 33%, 25%, 16%).

Enable SVG Uploads

Allow SVG files in your media library with automatic security sanitization. Dangerous elements are stripped on upload.

Duplicate Post Fields Tool

Copy fields between post types from the new Tools page. Select source, destination, and specific fields to duplicate.

Messenger Widget Improvements

New Persistent Admin Chat creates an always-visible chat circle for designated admin support. Plus per-widget default avatar uploads.

New Dynamic Tags

Access more data throughout your templates:

  • @post(vt_reviews.latest.content) – Latest review content, author, score, date, link
  • @post(vt_timeline.latest.*) – Same for timeline posts
  • @post(vt_wall.latest.*) – Same for wall posts
  • @post(article_helpful_latest_yes) – Last yes vote timestamp
  • @user(display_name).initial() – First letter with period ("J.")

Bug Fixes

We've squashed several bugs in this release:

  • Duplicate Post – Fixed blank page when viewing duplicated Voxel listings after publishing
  • Admin Columns – Fixed title display issues with Cyrillic characters showing as "..."
  • Admin Columns – Fixed post relations not showing related posts
  • Admin Columns – Author filter now includes all users who have posts, not just those with author role
  • Table of Contents Widget – Fixed avatar/profile fields displaying as "voxel:name" instead of proper labels
  • Table of Contents Widget – Fixed list style switching and real-time field completion updates
  • Active Filters Widget – Fixed filter removal not refreshing results and white space when empty
  • Suggest Edits Widget – Added option to show fields without values for more complete edit suggestions

Redesigned Settings Page

We've completely overhauled the settings page with tab-based navigation, making it easier to find and configure features as the toolkit grows.
