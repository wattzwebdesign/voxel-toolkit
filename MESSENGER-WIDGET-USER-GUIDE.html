Troubleshooting

Messages Not Sending?

  • Check your internet connection
  • Make sure the text box isn't empty
  • Try refreshing the page
  • Close and reopen the chat window

Chat Won't Open?

  • Confirm you clicked directly on the avatar
  • Close an existing chat if you've reached the maximum
  • Refresh your browser
  • Clear browser cache and try again

Badges Not Updating?

  • Scroll to view all messages in the conversation
  • Wait a few seconds for the update
  • Refresh the page
  • Check your connection status
